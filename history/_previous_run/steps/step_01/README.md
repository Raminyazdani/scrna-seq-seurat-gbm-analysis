# Single-Cell RNA-Seq Analysis Pipeline

## Overview

This project implements a comprehensive pipeline for analyzing single-cell RNA sequencing data, with a focus on glioblastoma tumor heterogeneity.

## Goals

- Quality control of scRNA-seq data
- Cell type identification
- Differential expression analysis
- Pathway enrichment analysis

## Tech Stack

- R
- Seurat
- Bioconductor packages

## Status

Project in initial setup phase.
